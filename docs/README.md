## Typische Einstiege

### 👨‍🔧 Ich bin Werkstatt-Inhaber und will verstehen, was das System kann
→ [Präsentation](./PRAESENTATION.md) durchblättern · Slides 1–15

### 💻 Ich bin Techniker und soll die Installation machen
→ [Schnellstart](./SCHNELLSTART.md) Schritt für Schritt durchgehen

### 🧑‍💼 Ich bin neuer Mitarbeiter und muss das System benutzen
→ [Handbuch Kapitel 5–9](./HANDBUCH.md#5-die-benutzeroberfläche) lesen (UI, Dashboard, Kalender, Termine, Kunden)

### 🎯 Ich führe es einem Kunden vor
→ [Präsentation](./PRAESENTATION.md) + vorher Demo-Daten laden (siehe [Kap. 20 im Handbuch](./HANDBUCH.md#20-demo-daten--reset))

### 🛠️ Ich integriere per API (eigene App, Telefon-KI, …)
→ [Handbuch Kap. 17](./HANDBUCH.md#17-schnittstellen--api-für-externe-systeme) und [Kap. 18](./HANDBUCH.md#18-telefon-ki-integration)

### 📑 Ich bin Steuerberater/Buchhalter und brauche Exporte
→ [Handbuch Kap. 22 (DATEV-Anbindung)](./HANDBUCH.md#22-datev-anbindung) + [Kap. 12 (Buchhaltung)](./HANDBUCH.md#12-buchhaltung)

## Was das System kann – in 30 Sekunden

```
Terminverwaltung     │ Dokumente           │ Finanzen
──────────────────   │ ──────────────────  │ ──────────────────
• Kalender (Tag/     │ • Angebote          │ • Einnahmen
  Woche/Monat)       │ • Rechnungen        │ • Ausgaben
• Drag & Drop        │ • Stornos           │ • Charts
• Dashboard          │ • Gutschriften      │ • CSV-Export
• Live-Betrieb       │ • Druckansicht      │ • Monats-Details
• Auslastungs-       │ • Aus Termin        │ • Kategorien
  Heatmap            │   erstellen         │

Stammdaten           │ Automatisierung     │ Integration
──────────────────   │ ──────────────────  │ ──────────────────
• Kunden             │ • E-Mail-Erinnerung │ • Online-Buchung
• Fahrzeuge          │ • SMS               │   (WordPress iFrame
• Historie           │ • WhatsApp          │    + JS-Widget)
• 140 Leistungen     │ • KI-Scan           │ • REST-API
• Bühnen             │   (Fahrzeugschein)  │ • API-Keys + Scopes
• Dienstpläne        │ • Cron-Jobs         │ • Webhooks
• Skills             │ • Telefon-KI        │ • CSV-Import
                     │                     │ • DATEV-Export
```
